# Minikube Namespace Exploration

## Objective
To set up Minikube locally and explore Kubernetes namespaces using kubectl.

## Tech Stack
- WSL (Ubuntu)
- Docker
- Minikube
- kubectl
- VirtualBox (installed on host system)

## Implementation Details
Minikube was executed inside WSL Ubuntu.
Due to VirtualBox driver limitations with WSL, the Docker driver was used as recommended in official Minikube documentation.

## Tasks Performed
- Started Minikube cluster
- Verified Kubernetes node status
- Created, listed, described, and deleted namespaces
